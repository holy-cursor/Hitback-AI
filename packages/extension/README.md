# HitBack — AI Wait State Ads

Earn money while your AI coding agent works. HitBack shows a small sponsored panel during active agent wait-states in **VS Code** and **Cursor**. You keep **60%** of ad revenue.

## How it works

1. Install the extension
2. Run **HitBack: Login** (Google sign-in via browser)
3. Use Cursor's AI agent as usual — when the agent is actively editing, a sponsored ad may appear
4. Track earnings and withdraw at [hitback.xyz/developer.html](https://hitback.xyz/developer.html)

HitBack never reads your source code, file contents, or chat prompts.

## Commands

| Command | Description |
|---------|-------------|
| `HitBack: Login` | Sign in to link impressions to your earnings account |
| `HitBack: Logout` | Sign out |
| `HitBack: Open Ad Link` | Open the current ad URL (also triggered by clicking the ad) |

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `hitback.enabled` | `true` | Turn ads on or off |
| `hitback.backendUrl` | `https://api.hitback.xyz` | HitBack API base URL |

## Privacy

- Anonymous install ID for impression counting
- Optional auth token stored in VS Code Secret Storage
- No document content or prompts are collected

See [Privacy Policy](https://hitback.xyz/privacy.html) and [Terms](https://hitback.xyz/terms.html).
